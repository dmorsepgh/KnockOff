# AI Views and News — Show Runner Business Plan
## v0.1.0 | March 31, 2026

---

## The Product

A turnkey AI news show production service for Skool community owners. We produce a branded, professional video news show tailored to any niche — delivered twice a month, ready to post.

**Deliverables per episode:**
- 8-15 min video show (intro bumper, host avatar, 3 news segments, credits)
- Written newsletter (Crypt Notes)
- Host notes with Q&A topics for live sessions
- All source links and story briefs

---

## Revenue Model

### Primary: Barter for Access
- Trade 2 shows/month for free community membership
- Zero cash exchanged — value for value
- Goal: access to every Skool community in our network
- Each community we join = new audience, new network, new leads

### Secondary: Paid Production ($50/episode)
- For communities that prefer to pay
- $50 per episode, 2 episodes/month = $100/mo per client
- Client picks 3 topics, we produce everything else

## Growth Strategy

### Phase 1: Proof of Concept (Now)
- Produce for Drake & Carter's community
- Refine the workflow and show format
- Build 3-5 sample episodes across different niches

### Phase 2: Barter Network (Month 2-3)
- Approach 5-10 Skool community owners with the pitch video
- Offer barter deal: free show for free membership
- Build a portfolio of shows across niches (crypto, fitness, real estate, etc.)

### Phase 3: Paid Clients (Month 3-6)
- Use barter portfolio as proof/demos
- Offer $50/episode to communities outside the barter network
- Target: 5 paid clients = $500/mo

### Phase 4: Scale (Month 6+)
- Invest in API automation (HeyGen + Pictory)
- Each episode drops to 10 min production time
- Target: 10-20 clients (mix of barter + paid)
- Consider hiring a virtual assistant for copy/paste work before committing to API costs

## Competitive Advantage

1. **Near-zero marginal cost** — tools are flat-rate, not per-video
2. **Speed** — 60 min per episode, not days
3. **Show Runner platform** — web-based, shareable, collaborator-friendly
4. **AI-native** — scripts, summaries, Q&A topics all auto-generated
5. **Niche-agnostic** — swap RSS feeds, same pipeline works for any topic
6. **Barter model** — competitors charge cash, we trade value

## Risks

| Risk | Mitigation |
|------|-----------|
| HeyGen/Pictory price increases | KnockOff (local) as long-term backup |
| Free tier Pictory gets limited | Switch to paid ($25/mo) or API |
| Quality concerns from clients | Episode review step before delivery |
| Too many clients, not enough time | API automation or hire VA |
| Platform TOS changes | Diversify tools, keep local alternatives |
